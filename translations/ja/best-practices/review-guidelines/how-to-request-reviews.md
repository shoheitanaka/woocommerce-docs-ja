# How to request WooCommerce extension reviews

## レビュー依頼の方法

### 管理者からのお知らせ

管理者通知は、レビューを要求する業界標準の方法ですが、管理者ダッシュボードに管理者通知を浴びせるのは効果的ではありません。通知のデザインは控えめにし、一度に1つの通知にとどめることをお勧めします。あまりに多くの通知や押し付けがましい通知でマーチャントを圧倒するのはとても簡単です。

#### 推薦の言葉

* 拡張機能をレビューするための管理者向け通知は、`Plugins`ページと拡張機能の設定ページに表示するのがよいでしょう。
* 通知にはスヌーズオプション（または複数）を設け、いつ通知が再表示されるかを明確にしましょう。
* 管理者通知は、常に完全に解除できるようにしてください。スヌーズオプションだけではいけません。
* 通知で提示されるオプションは、操作的な表現を避けるために注意深く表現されなければならない。
* 一貫したデザインの通知を使用することで、レビューのリクエストが拡張機能の一部のように感じられ、WooCommerceのデザインと一貫しているように見えます。

### 直接接触

#### 推薦の言葉

* 好意的である可能性が最も高いレビューを依頼する最も直接的な方法は、顧客が製品に最も満足しているときに連絡を取ることです。
* これは、以下のタイミングガイドラインに従って、マイルストーンでも時間ベースでも構いません。
* このための最良の方法は、電子メールまたはその他の直接的なやり取り（サポートチャット、電話など）です。これはコンバージョン率が最も高く、特に顧客が最も満足するように適切なタイミングを計った場合に有効です。
* これはまた、あなたのプラットフォームで一定量の処理を行ったマーチャントや、フルフィルメントエクステンションを使用して最初の100件の注文を発送したマーチャントなど、特に適格なマーチャントからのフィードバックを要求できる場合にも非常に効果的です。
* 直接アウトリーチが成功する可能性が高いのは、レビュー依頼をするユーザーをターゲットにする方法（マーチャントアカウント/利用情報など）と、営業チームやマーケティングチームがマーチャントにEメール/電話/チャットで問い合わせるなど、レビューを収集する方法がある場合です。

## レビュー依頼用メッセージ

私たちが推奨するフィードバックを求める方法のひとつは、NPSスタイルのレビュー募集を利用することです。これにより、肯定的なレビューを増やすだけでなく、苦戦している加盟店を支援する機会を提供することができます。

NPSスタイルのレビューは、まずユーザーに製品の評価（5つ星のうち）を尋ね、その回答に基づいてルーティングを行う：

* 4つ星または5つ星をクリックしてもらったら、レビューを残してもらう。
* 1,2,3のいずれかをクリックした場合は、私たちがお手伝いすることを伝え、サポートチケットを提出してもらう。

商品購入者は、サポート担当者と積極的なやり取りをした後にレビューを残す可能性が著しく高い。私に感謝する最善の方法は、その中で私について言及した5つ星のレビューを残すことです "または同様の言葉は非常にうまく機能する傾向があります - 人々は、生産者や企業よりも個人を助けたいと思っています。
