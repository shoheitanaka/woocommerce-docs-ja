# レビューガイドライン

レビューはオンラインショッピング体験に不可欠な要素であり、ソフトウェアをインストールする人々はレビューに注意を払います。あなたのエクステンションの見込みユーザは、ソフトウェアを選択する際に平均的な評価を考慮するでしょう。

YelpのビジネスレビューからAmazonの製品レビューまで、今日最も人気のあるオンラインレビュープラットフォームの多くは、極端に肯定的または否定的なレビューが多く、穏健な意見が少ない、二極化する可能性のある意見の範囲を持っています。このため、レビューの分布が「J字型」になり、正確さや有用性が損なわれてしまうのです。

WooCommerce.comとWordPress.orgはどちらもレビューを大きく取り上げており、より高い評価を得ている競合エクステンションの方が、ユーザーの選択において優位に立っている可能性が高い。 

## レビューにまつわる主な考慮事項

また、既存のレビューを分析することで、レビューを募集する前に対処すべき点を浮き彫りにすることができる。

WordPressのプラグインのユーザーがオーガニックにレビューを残すことは極めて稀であり（WordPress.orgのユーザーのうちレビューを残すのは0.2％）、平均的なプラグインのユーザーの99.8％に未開拓の市場があることを意味する。

これらのプラグインは、WordPress.org のプラグインディレクトリで同じ検索キーワードで他のプラグインと競合しており、評価はランキングアルゴリズムの大きな要因となっています。これは通常、WooCommerce Marketplaceでは同じ程度の要因ではありません。例えば、WooCommerceのPayPalエクステンションは、WordPress.orgリポジトリ上の他のPayPalエクステンションとあらゆるキーワードで直接競合しますが、マーケットプレイス上の他のPayPal決済エクステンションとは競合しません。
