# 共有可能なチェックアウトURL

カスタムチェックアウトリンクは、特定の商品を自動的にカートに入れ、ユニークなセッションIDで顧客をチェックアウトに直接リダイレクトします。 

カスタム・チェックアウト・リンク・パスは`/checkout-link/`で、カスタマイズできません。

## パラメーター

### 製品

```plaintext
products=123:2,456:1
```

商品IDと数量のカンマ区切りリスト。例えば、`123:2,456:1`。この機能は、追加オプションのないシンプルな商品をサポートします。正しいバリエーションIDを使用することで、個々のバリエーションをカートに追加することもできます。

### Coupon

```plaintext
coupon=SPRING10
```

カートに適用するクーポンコード。例えば、`SPRING10`。

## Example

```plaintext
https://yourstore.com/checkout-link/?products=123:2,456:1&coupon=SPRING10
```

このリンクで

- 商品ID `123` に数量 `2` が追加されます。
- 商品ID `456` に数量 `1` が追加されます。
- クーポンコード `SPRING10` が適用されます。
- 顧客は直接チェックアウトページに移動します。

## セッション

ユーザーがチェックアウトページにリダイレクトされると、カートに商品とクーポンコードが入力されます。最終的なURLには、セッションのIDを保存する`session`パラメータが含まれます。チェックアウトへの今後の変更はセッションに永続化され、永続的で共有可能なカートが可能になります。
