# WooCommerce CLIの概要

## WooCommerce CLI入門

WooCommerce CLI (WC-CLI)はコマンドライン経由でWooCommerce (WC)ストアを管理する効率的な方法を提供します。このツールはバージョン3.0.0以降のWCの一部であり、WC REST APIの機能を活用しています。つまり、REST APIで実現可能なほとんどのタスクはコマンドラインでも実行可能です。

WCバージョン2.5と2.6のCLIに関するドキュメントは、[Legacy CLI commands](https://github.com/woocommerce/woocommerce/wiki/Legacy-CLI-commands-(v2.6-and-below))をご覧ください。

## WP-CLIについて

WP-CLIは、WordPressのインストールを管理するための強力なコマンドラインツールのセットです。プラグインの更新やマルチサイトの設定など、ウェブブラウザを使わずにさまざまなことができます。詳細については、[WP-CLI公式ウェブサイト](http://wp-cli.org/) をご覧ください。

## WooCommerce CLIを始めよう

WC-CLIを使い始めるには、WP-CLIがインストールされていることと、WooCommerce 3.0.0以降が動作していることを確認してください。CLIコマンドは`wp wc`コマンドでアクセスできます。利用可能なコマンドの完全なリストは、コマンドラインインターフェイスで`wp wc`と入力してください。

コマンドとその使用方法の詳細については、コマンドリファレンス セクションを参照してください。ハウツーガイドは、WC-CLIを初めて使用する方や、迅速な解決策をお探しの方に最適な、一般的なタスクに対する実践的なアプローチを提供します。
