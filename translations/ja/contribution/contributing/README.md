# ♪ ウーに貢献しよう

Wooコントリビューターコミュニティへの参加に興味がありますか？このドキュメントのリンクは、あなたが最初の貢献をするために必要な操作の順序を要約し、案内しています。経験豊富なWooCommerce開発者の方は、テンプレートリンクや以下のリファレンスドキュメントやガイドをご利用ください。

## WooCommerceコアへの貢献

WooCommerceコアプラグインコードは[monorepo](https://github.com/woocommerce/woocommerce)にあります。ここに貢献できます： 

- [WooCommerceコアプラグイン](https://github.com/woocommerce/woocommerce/tree/trunk/plugins/woocommerce)
- [WooCommerceブロック](https://github.com/woocommerce/woocommerce/blob/trunk/plugins/woocommerce/client/blocks) 

### 行動規範

オープンソースプロジェクトに貢献するためには、私たちのプロジェクトを安定かつ安全に構築し、質問できる場所にするために、個人や組織の協力が必要です。私たちの[行動規範](https://github.com/woocommerce/woocommerce/blob/trunk/SECURITY.md)を熟読し、私たちの基準とプロセスに精通してください。

### 投稿者ガイドライン

私たちの[投稿者ガイドライン](https://github.com/woocommerce/woocommerce/blob/trunk/.github/CONTRIBUTING.md)は、投稿への最初のステップをレイアウトしています。

- [始めるための前提条件と開発者ツール](https://github.com/woocommerce/woocommerce/blob/trunk/README.md#getting-started)
- [PNPMコマンド、プラグイン開発環境パッケージ、トラブルシューティング](https://github.com/woocommerce/woocommerce/blob/trunk/DEVELOPMENT.md)
- [コーディング標準、E2Eテストのリンク](https://github.com/woocommerce/woocommerce/blob/trunk/.github/CONTRIBUTING.md) 

### テンプレート、バグ報告、機能リクエスト

- [プルリクエストテンプレート](https://github.com/woocommerce/woocommerce/blob/trunk/.github/PULL_REQUEST_TEMPLATE.md)
- [コア課題テンプレート](https://github.com/woocommerce/woocommerce/blob/trunk/.github/ISSUE_TEMPLATE.md)
- [バグ報告テンプレート](https://github.com/woocommerce/woocommerce/blob/trunk/.github/ISSUE_TEMPLATE/1-bug-report.yml)

- [機能リクエスト](https://woocommerce.com/feature-requests/woocommerce/)

### セキュリティ

[データ管理のセキュリティと安全性はWooにとって非常に重要です。Automatticセキュリティポリシー](https://automattic.com/security/)をご覧ください。

Wooの[セキュリティポリシー](https://github.com/woocommerce/woocommerce/blob/trunk/SECURITY.md)をお読みになり、脆弱性やセキュリティ上の問題を報告してください。
