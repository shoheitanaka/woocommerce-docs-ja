# Payment Button Layout

コンテナに対するボタンの位置と配置を定義し、ボタン間の階層と間隔を均等にします。

## 商品ページ

横長または縦長のレイアウトで「カートに入れる」ボタンに対して支払いボタンを正しく配置し、支払いボタンを「カートに入れる」ボタンの右側または下に配置する。

![商品ページレイアウト.png](https://github.com/shoheitanaka/woocommerce-docs-ja/tree/main/docs/_docu-tools/static/img/Product-page-layout.png)

縦型レイアウトの「チェックアウトに進む」ボタンに対して、支払いボタンを正しく配置する。決済ボタンは「チェックアウトに進む」ボタンの上に配置してください。支払いボタンはボタンの下や間に配置しないでください。

![カートレイアウト.png](/img/doc_images/カートレイアウト.png)

## エクスプレス・チェックアウト

支払いボタンは水平レイアウトで正しく配置し、サイズと間隔を統一する。2列目が必要な場合は、ボタンを左揃えにする。

![エクスプレス・チェックアウトのレイアウト.png](/img/doc_images/Express-checkout-layout.png)

## モバイルビュー

モバイルのエクスプレス決済ボタンは、横幅いっぱいに配置してください。エクスプレス決済の枠線は表示領域を狭めるので使用しないでください。

![モバイルビュー.png](/img/doc_images/モバイルビュー.png)

## クリアスペース

支払いボタンの四方に最小限のクリアスペースを確保してください。クリアスペースはボタンの大きさによって変わります。

![クリアスペース.png](/img/doc_images/Clear-space-updated.png)
