# 商品編集者のガイドライン

私たちの調査によると、Wooで新しい商品を作成する際、マーチャント様はタスクとゴールという観点で考えています。例えば、商品価格の追加や編集は、画像のアップロードや在庫の管理とは別のタスクです。

このため、新しい製品フォームには、これらのタスクに対応するグループ（タブ）が用意されています。簡単に言えば、異なるフォーム機能が存在する別々のビューです。開発者として、これらのビューのいずれかを拡張し、より良い、より成功する製品を作成するために必要な情報やツールを加盟店に提供することができます。

新商品フォームを拡張する方法はいくつかあります: 単一のフィールドから、複数のフィールドやテーブルを含むグループやセクション全体まで。これらの拡張ポイントはフォーム構造にリンクされており、素晴らしいユーザーエクスペリエンスを創造する自由を与えてくれます。
