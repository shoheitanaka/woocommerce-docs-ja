# Tools for low-code development

#はじめに

このガイドでは、Gutenberg や WooCommerce Store Editing などのローコードソリューションのほか、カスタムコーディングなしで WooCommerce ストアを作成するためのページビルダーやビルド済みコンポーネントを紹介します。これらのツールを理解し活用することで、ローコードビルダーは最小限のコーディングでストアを構築し、Eコマースウェブサイトのデザインとユーザーエクスペリエンスの側面に集中することができます。

## 観客

このガイドは、ローコードビルダー、またはWordPressとWooCommerceの基本的な知識を持ち、大規模なカスタムコーディングを必要とせずにオンラインストアを作成したい人を対象としています。

## 前提条件

このガイドに従うには、以下のものが必要である：

1.WordPressとWooCommerceの基本的な理解。
2.WooCommerceがインストールされ、有効化されたWordPressウェブサイト。

## ステップ1 - GutenbergとWooCommerceストアの編集を使う

GutenbergはWordPressのデフォルトのブロックエディタで、ブロックを追加・カスタマイズしてページを作成・編集することができます。WooCommerceはGutenbergの機能を拡張し、商品ページやショップページなどWooCommerce特有の要素を作成・カスタマイズできるようになりました。

GutenbergとWooCommerceストア編集を使用する：

1.WordPressのインストールが最新で、Gutenbergが最新バージョンであることを確認してください。
2.ブロックテーマをインストールして有効化し、WooCommerceの店舗編集機能を有効にします。

GutenbergとWooCommerceストア編集を使えば、テキスト、画像、ボタン、商品グリッドやショッピングカートなどのWooCommerce特有のブロックなど、様々なブロックを使ってストアのページを作成・カスタマイズすることができます。

## ステップ2 - 代替ページビルダーの検討

GutenbergとWooCommerce Store EditingはローコードでWooCommerceストアを構築するための強力なオプションですが、より高度な機能や特定のユースケースのために他のページビルダーの使用を検討することもできます。WooCommerce と互換性のある一般的なページビルダーには次のようなものがあります：

1.エレメンタ
2.ビーバービルダー
3.Diviビルダー
4.WPBakeryページビルダー

あなたのニーズと予算に合ったページビルダーを選択し、WordPress ウェブサイトにインストールして有効化します。これらのページビルダーは、通常、カスタムコードを記述することなく、完全に機能するWooCommerceストアを作成するために使用することができ、事前に構築されたコンポーネントのライブラリを提供しています。

## ステップ3 - 構築済みのコンポーネントとテンプレートを利用する

Gutenbergを含む多くのページビルダーには、ページに簡単に追加できるコンポーネントやブロックがあらかじめ用意されています。これらのコンポーネントには、ボタン、フォーム、画像ギャラリーなどのデザイン要素や、商品グリッドやショッピングカートなどのWooCommerce固有のコンポーネントが含まれます。

さらに、いくつかのページビルダーやWooCommerceエクステンションは、あなたがインポートし、迅速に完全に機能するオンラインストアを作成するためにカスタマイズすることができ、事前に構築されたストアテンプレートを提供しています。これらのテンプレートは、あなたのストアのためのプロフェッショナルなデザインの出発点を提供することで、時間と労力を節約することができます。

構築済みのコンポーネントやテンプレートを使用する：

1.お好みのページビルダーのエディター（Gutenbergまたは他のページビルダー）を開く。
2.利用可能なコンポーネント/ブロックまたはテンプレートをブラウズして、あなたのニーズに合ったものを見つけてください。
3.コンポーネントをページに追加し、提供されている設定やオプションを使用してカスタマイズします。

## 結論

Gutenberg、WooCommerce Store Editing、その他のページビルダーのようなローコードソリューションを活用することで、カスタムコーディングの必要なく、完全に機能するWooCommerceストアを作成することができます。このガイドでは、これらのツールの基本的な使い方を紹介し、利用可能なオプションを理解し、最小限のコーディングでストアを構築できるようにします。ツールとテンプレートの適切な組み合わせにより、あなたのビジネスニーズを満たす、プロフェッショナルでユーザーフレンドリーなEコマースサイトを作成することができます。
